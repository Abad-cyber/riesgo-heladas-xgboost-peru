# riesgo-heladas-xgboost-peru
Proyecto de aprendizaje automático para la predicción del riesgo de heladas en regiones altoandinas del Perú utilizando datos meteorológicos y el algoritmo XGBoost.

## Reproducibilidad
Los datos utilizados no se incluyen directamente en este repositorio debido a su tamaño.
Para reproducir los experimentos, descargue el conjunto de datos desde Kaggle y colóquelo en el directorio de trabajo:

https://www.kaggle.com/datasets/abadpomamaquera/meteorological-data-for-frost-prediction-in-peru
